Project title: Investigating Gender Bias in Graduate Admissions at UC Berkeley
Description: In this causal analysis, we examine whether gender bias existed in graduate school admissions at University of California, Berkeley in 1973
Research question: Did gender causally affect admission outcomes at UC Berkeley in 1973?
Methods employed: OLS, fixed effects, Logistic regression, causal inference, data visualization.
Key findings: The striking finding of the analysis is the absence of gender-based discrimination in the admissions process. We observed aggregate gap is instead driven by application patterns, as women disproportionately applied to more competitive programs compared to men.
Tools: Python
